# WST_DAS
A kind of water surface target data detected by distributed acoustic sensing technology
一种用分布式声波传感技术探测的水面目标数据
A kind of water surface target data detected by distributed acoustic sensing technology(WST_DAS)

WST_DAS数据说明：
每个mat里面是100*50000的矩阵,每个mat采集时长5秒，数据为阵列采集的未去直流分量的原始数据
阵列有100个传感器通道，每个传感器相邻10米，阵长1000米，采样率为10000Hz

20250811094242_mat
工况为水面船4节速度垂直穿过阵列中心,开始穿阵时大约离阵中心420米，结束穿阵时大约离阵中心325米；

20250811100902_mat
工况为水面船12节速度以45度斜着穿过阵列中心,开始穿阵时大约离阵中心760米，结束穿阵时大约离阵中心390米；
